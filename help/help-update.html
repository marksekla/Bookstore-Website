Updating Website Content

This guide explains how to update the Bookworm Haven website content, including adding new books, changing images, and modifying the catalog. This guide is intended for website administrators and content managers.


Updating the Book Catalog

The book catalog is stored in a JSON file, which makes it easy to update without programming knowledge. Follow these steps to add, modify, or remove books from the catalog:


Locating the Books JSON File

The book catalog is stored in the file data/books.json. You can edit this file using any text editor such as Notepad, TextEdit, or Visual Studio Code.


Understanding the JSON Structure

Each book in the catalog is represented as a JSON object with the following properties:

{
    "title": "The Great Gatsby",
    "author": "F. Scott Fitzgerald",
    "category": "Classic",
    "price": 10.99,
    "image": "book1.jpg"
}

Adding a New Book

To add a new book to the catalog:

  1. Open the books.json file in a text editor
  2. Locate the closing bracket ] at the end of the file
  3. Before the closing bracket, add a comma after the last book's closing curly brace }
  4. Add a new book entry following the JSON structure shown above
  5. Save the file

Modifying an Existing Book

To modify an existing book:

  1. Open the books.json file in a text editor
  2. Locate the book entry you want to modify
  3. Change the values as needed, keeping the JSON structure intact
  4. Save the file

Removing a Book

To remove a book from the catalog:

  1. Open the books.json file in a text editor
  2. Locate the book entry you want to remove
  3. Delete the entire book object, including the opening and closing curly braces
  4. Make sure to maintain proper JSON syntax by removing any trailing commas if needed
  5. Save the file

Important: Always make a backup of the books.json file before making changes. Make sure the JSON syntax is correct, or the website may not display the books properly.


Adding or Changing Book Images

Each book in the catalog has an associated image. Follow these steps to add or change book images:

Preparing Book Images

For best results, prepare book images with the following specifications:

  • Format: JPG or PNG
  • Recommended size: 400x600 pixels
  • Maximum file size: 500KB
  • Use copyright-free images or images you have permission to use

Adding New Book Images

  1. Name your image file appropriately (e.g., book21.jpg for the 21st book)
  2. Upload the image file to the images/ directory on the server
  3. Update the books.json file to reference the new image file name

Replacing Existing Book Images

  1. Create or prepare a new image file
  2. Either use the same filename as the existing image to automatically replace it, or
  3. Use a new filename and update the image property in the books.json file
  4. Upload the new image to the images/ directory on the server

Updating Media Files

The website includes audio and video files that can also be updated:

Replacing Media Files

  1. Prepare your new media file in the appropriate format (MP4 for video, MP3 for audio)
  2. Name it to match the existing file you want to replace
  3. Upload the file to the media/ directory on the server

Troubleshooting

If you encounter issues after updating content:

JSON Syntax Errors

If books don't appear after updating the JSON file:

  • Check for proper JSON syntax - all quotes, commas, and brackets must be correctly placed
  • Verify that all entries have commas between them, but no comma after the last entry
  • Use a JSON validator tool online to check your file

Image Issues

If images don't appear:

  • Verify that the image filename in the JSON file exactly matches the actual file (case-sensitive)
  • Check that the image is uploaded to the correct directory
  • Ensure the image is in a supported format (JPG or PNG)
